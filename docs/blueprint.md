# **App Name**: Krav Magá IPIRANGA Manager

## Core Features:

- Public Registration Form: A public-facing form for students to submit their initial data via a link.
- Admin Dashboards: Overview of active students, overdue payments, upcoming exams, and financial reports.
- Automated Payment Reminders: Automatically send payment reminders via email for overdue plans. Also provides a button, for manual whatsapp sending.
- Data storage: Store the student and training data into Firestore, and make backups every day

## Style Guidelines:

- Deep blue (#00008e) for a sense of discipline and focus, reflecting the martial arts theme.
- Background color: Very light lavender (#F0F8FF) provides a calm and uncluttered backdrop.
- Accent color: ANyone that adds depth without being too distracting, analogous to the primary color.
- Body and headline font: 'PT Sans', a modern, readable, humanist sans-serif, will provide an approachable user experience.
- Use clean and modern icons to represent data points and actions.
- Prioritize a clean and intuitive layout with clear data visualization for the admin dashboards.
- Subtle transitions and animations to enhance user experience when navigating between dashboards and data.
